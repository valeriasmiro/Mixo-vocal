# Как добавить афишу

1. Загрузите файл изображения (jpg/png/webp) прямо в эту папку `posters/`.
2. Откройте `posters.json` и добавьте имя файла в список.

Самый простой вариант — просто имя файла:

```json
[
  "koncert-iyun.jpg",
  "master-klass-dzhaz.png"
]
```

Если хотите задать своё название афиши (иначе оно возьмётся из имени файла),
используйте расширенный формат — можно смешивать оба варианта в одном списке:

```json
[
  "koncert-iyun.jpg",
  { "file": "master-klass-dzhaz.png", "title": "Джазовый мастер-класс" }
]
```

Больше ничего менять не нужно — афиша появится на главной странице
в разделе «Афиши» автоматически, в своей рамке, и откроется крупно
по клику с возможностью приблизить.
