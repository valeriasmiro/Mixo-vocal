# MIXO — сайт студии современного вокала

Статический одностраничный сайт: студия, абонементы/разовые занятия,
акции, расписание и адрес, а также лента афиш с просмотром крупным
планом.

## Структура

```
index.html          — вся разметка и текст сайта
css/style.css        — стили (палитра, типографика, вёрстка)
js/script.js         — меню, загрузка афиш из JSON, лайтбокс с зумом
posters/             — сюда добавляются картинки афиш
posters/posters.json — список афиш (имя файла + опционально title)
posters/README.md    — короткая инструкция по добавлению афиш
```

## Как открыть

Просто откройте `index.html` в браузере, либо включите GitHub Pages
для этого репозитория (Settings → Pages → ветка `main`, папка `/`).

## Что редактировать

Все места, которые стоит заполнить своими данными, отмечены в
`index.html` комментарием `<!-- РЕДАКТИРУЕТСЯ: ... -->`:

- **Абонементы** (раздел `pricing`) — названия, цены и описания карточек.
- **Акции** (раздел `promo`) — три карточки с текущими предложениями.
- **О студии** (раздел `about`) — текст описания и цифры.
- **Расписание и адрес** (раздел `schedule`) — часы работы, адрес, телефон, e-mail.

## Афиши

Чтобы добавить афишу на главную страницу: положите картинку в папку
`posters/` и впишите её имя в `posters/posters.json`. Подробности — в
`posters/README.md`. Пока список пуст, раздел показывает аккуратную
пустую рамку-заглушку с подсказкой.

## Палитра

- Кремовый фон: `rgb(245, 233, 211)`
- Насыщенный жёлтый (акцент): `rgb(253, 220, 5)`
- Глубокий чёрный: `#17140F`

> Кремовый цвет указан как `rgb(245, 233, 211)` — в исходном сообщении
> третье значение (311) выходило за допустимый диапазон 0–255, поэтому
> оно было приближено до ближайшего похожего тёплого кремового оттенка.
> Если нужен другой оттенок — значения заданы переменной `--cream` в
> начале `css/style.css`, их легко поменять в одном месте.

## Шрифты

- Заголовки: **Fraunces** (курсив/италик для акцентов — отсылка к
  театральным афишам)
- Текст: **Manrope**

Подключены через Google Fonts в `<head>` файла `index.html`.
